[[req_core_process-execute-inputs]]
[requirement]
====
[%metadata]
label:: /req/core/process-execute-inputs
[.component,class=part]
--
The server SHALL support process input values specified in-line in an execute request (i.e. by value).
--

[[input_by_reference]]
[.component,class=part]
--
The server SHALL support process input values specified by reference (i.e. using a http://schemas.opengis.net/ogcapi/processes/part1/1.0/openapi/schemas/link.yaml[link]).
--
====
