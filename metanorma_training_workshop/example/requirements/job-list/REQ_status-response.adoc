[[req_job-list_status-response]]
[requirement]
====
[%metadata]
label:: /req/job-list/status-response

If the parameter is specified with the operation, only jobs that have a value for the `status` property (see: http://schemas.opengis.net/ogcapi/processes/part1/1.0/openapi/schemas/statusInfo.yaml[statusInfo.yaml]) that matches one of the specified values of the `status` parameter SHALL be included in the response.
====
